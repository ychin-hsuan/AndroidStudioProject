## ▶️ 如何執行

1. **Clone 此 Repository**
   ```bash
   git clone https://github.com/ychin-hsuan/AndroidStudioProject.git
   ```

2. **開啟 Android Studio**，選擇 `File > Open`，選取想執行的子資料夾（例如 `AndroidStudio_HW1`）。

3. **等待 Gradle 同步完成**。

4. 連接 Android 實體裝置或啟動模擬器（AVD Manager），點擊 ▶️ **Run** 執行應用程式。

> ⚠️ 注意：每個子資料夾為獨立的 Android Studio 專案，請個別開啟，不要直接開啟根目錄。
